# Pythagorean Tree Generator

## Overview

This Python project uses OpenCV to generate Pythagorean tree squares, creating a visual representation with sky, earth, moon and stars ,using OpenCV.

## Libraries

1. OpenCV.
2. NumPy

## Requirements

- Python 3.x
